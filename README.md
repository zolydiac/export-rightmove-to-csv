# Export Rightmove to CSV

Export property listings from Rightmove search results into a clean CSV you can open in Excel or Google Sheets.

## What it does
- Works on Rightmove search results pages (for sale and to rent)
- Scans the listings on the page youâ€™re viewing
- Exports a CSV file to your Downloads folder
- Lets you preview data and choose which columns to include

## How to use
1. Open a Rightmove results page (for sale or to rent).
2. Click the extension icon in Chrome.
3. Click **Scan** to detect listings.
4. (Optional) Toggle columns / check the preview.
5. Click **Export CSV**.

## Free vs Pro
- **Free:** limited exports per day and limited rows per export.
- **Pro:** unlimited exports and unlimited rows.

Pro is activated via ExtensionPay after checkout.

## Privacy
- Listing data is processed **locally in your browser**.
- No Rightmove listing data is uploaded to any server.
- The extension only contacts ExtensionPay to check Pro status (license verification only).
